# [CodeGig](https://13pz9.sse.codesandbox.io/ "Live Demo")

### This app uses :
- Node.js
- Express.js
- Sequalize 
- Postgres
- Handlebars.js
- Body Parser

## Quick Start

``` bash
# Install dependencies
npm install

# Serve on localhost:5000
npm start
# Dev Server (Nodemon)
npm run dev
```
